## AI 动态

### OpenAI 与 Broadcom 联合发布 Jalapeño 推理芯片

OpenAI 正式推出与 Broadcom 合作开发的 LLM 优化推理芯片 Jalapeño。

- **原文链接**：https://openai.com/index/openai-broadcom-jalapeno-inference-chip
- **我的判断**：OpenAI 不满足于只做模型层，正在向下游基础设施延伸。这跟 Google TPU、AWS Trainium 是同一逻辑——通过软硬一体优化降低推理成本。Jalapeño 如果能大规模量产，可能显著降低 GPT 系列模型的 API 价格。
- **对你的影响**：短期无直接影响。中长期关注 OpenAI API 降价趋势，降价会降低 Agent 和 AI 应用的边际成本 diket。

### GPT-5 助力免疫学突破

GPT-5 帮助免疫学家 Derya Unutmaz 解决了一个 3 年未解的免疫学难题。

- **原文链接**：https://openai.com/index/gpt-5-immunology-mystery
- **我的判断**：这是 GPT-5 在科学研究中应用的又一案例。与前几周的"AI 化学家"、"LifeSciBench"形成连续叙事——OpenAI 在强化"AI for Science"的定位。
- **对你的影响**：AI 在垂直领域的渗透不可逆。作为工程师，了解 AI 在不同行业的落地模式有助于判断技术趋势。
