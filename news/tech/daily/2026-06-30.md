## Java 动态

### ZGC：十年重新定义 Java 性能

Inside Java 发布 ZGC 十年回顾文章。

- **原文链接**：https://inside.java/2026/06/30/zgc-performance-decade/
- **我的判断**：ZGC 是 Java GC 发展的里程碑。亚毫秒级暂停的 GC 让 Java 在低延迟场景下具备了与 C++/Go 竞争的能力。十年来 ZGC 从实验性特性成长为生产就绪的 GC 引擎。
- **对你的影响**：如果你的项目对延迟敏感（如交易系统、实时分析），ZGC 是必须掌握的 JVM GC 选项。
