# 技术报告

本分类用于沉淀与你的核心关注方向相关的技术情报与工程分析，重点包括：

- AI / Agent / MCP
- OpenAI / Anthropic / Gemini / DeepSeek / Qwen
- Java / Spring / Spring Boot / Gradle / Maven / JOOQ
- PostgreSQL / Docker / Kubernetes / DevOps
- 架构设计 / 系统设计 / 开源项目 / 安全漏洞

下设两个固定子类：

- `daily/`：技术日报
- `weekly/`：技术周报
